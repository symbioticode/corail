# Module 3: Détection des Cycles Fractals
## Objectif
Identifier les cycles fractals dans les données de marché pour optimiser le timing des trades.
## Fichiers
- cycle_detection.py: Détection des cycles pour paires natives.
- cycle_detection_adjusted.py: Détection ajustée pour stablecoins.
## Exemple
Consultez examples/cycle_btc_eth_vs_usdt.ipynb pour une analyse des cycles.
## Clé Éthique
**Résilience**: Priorisez la stabilité systémique.
