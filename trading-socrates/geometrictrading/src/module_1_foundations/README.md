# Module 1: Fondations de la Géométrie Émergente
## Objectif
Introduire la **Théorie de la Géométrie de l'Émergence (TGE)** et le score ERQ pour modéliser les relations entre actifs financiers dans l'espace QAAF 4D (momentum temporel, volatilité directionnelle, phase spectrale, entropie contextuelle).
## Fichiers
- erq_calculation.py: Calcule le score ERQ pour les paires natives (ex. BTC/ETH).
- erq_comparison.py: Compare le score ERQ pour paires natives vs stablecoins (ex. BTC/USDT).
## Exemple
Consultez examples/erq_btc_eth_vs_usdt.ipynb pour un exemple d'analyse.
## Clé Éthique
**Transparence**: Déclarez si les analyses utilisent des paires natives ou stablecoins.
