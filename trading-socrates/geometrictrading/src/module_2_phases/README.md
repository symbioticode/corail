# Module 2: Classification des Phases Géométriques
## Objectif
Classifier les phases géométriques (Accumulation, Expansion, Climax, Distribution, Neutral) pour guider les décisions de trading.
## Fichiers
- phase_classification.py: Classification pour paires natives.
- phase_classification_adjusted.py: Classification ajustée pour stablecoins.
## Exemple
Consultez examples/phase_btc_eth_vs_usdt.ipynb pour une analyse comparative.
## Clé Éthique
**Non-Exploitation**: Évitez les stratégies spéculatives déstabilisantes.
