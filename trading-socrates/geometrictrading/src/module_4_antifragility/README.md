# Module 4: Anti-Fragilité et Tenseur Symbiotique
## Objectif
Construire des portefeuilles anti-fragiles à l'aide du tenseur symbiotique.
## Fichiers
- symbiotic_tensor.py: Calcul du tenseur pour paires natives.
- symbiotic_tensor_adjusted.py: Calcul ajusté pour stablecoins.
## Exemple
Consultez examples/tensor_portfolio_btc_eth.ipynb pour un portfolio anti-fragile.
## Clé Éthique
**Diversité**: Promouvez des allocations diversifiées.
